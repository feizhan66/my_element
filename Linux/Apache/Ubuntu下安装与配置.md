
安装apache2
```bash
sudo apt install apache2
```

安装php
```bash
sudo apt-get install php7.0 libapache2-mod-php7.0
```
重启apache2
```bash
sudo /etc/init.d/apache2 restart
```
安装phpmyadmin
```bash
apt-get install phpmyadmin
# 默认安装到 /usr/share/phpMyAdmin 
```
